## 二层交换机配置
- enable root用户
- config terminal 配置模式
	- vlan xx 进入vlan xx 配置模式，不存则则顺便创建
	- interface fa0/1 进入fa0/1 端口配置模式
		- switch\[port] access vlan xx 将vlan xx 绑定到端口上
		- switch mode trunk 改变端口模式
		- swtich \<mode> allowed vlan all 再指定模式下允许所有vlan通过
